## HSV色彩空间
在HSV彩色空间中（色相/饱和度/明度（Hue/Saturation/Value）），遥感影像阴影区域与非阴影区域相比有以下3 个特点：
- 阴影区域具有更大的色调值;  
- 阴影区域的散射光线主要来自波长更短的蓝紫色光，因此具有高饱和度值;  
- 阴影区域太阳光线被阻挡，导致低亮度值. 
  
    将彩色影像进行ＲGB 到HSV色彩空间变换，依据阴影区域的高色调值、低亮度值和高饱和度特性，定义M= ( S－V) /( H+S+V) 进行阈值选择分割出阴影区域。
